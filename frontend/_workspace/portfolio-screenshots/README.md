# 포트폴리오 스크린샷 정리

## 추천 사용 순서

1. `selected/portfolio-01-overall-screens.png`  
   전체 화면 구성 소개용입니다. 포트폴리오 첫 이미지나 기능 목록 섹션에 적합합니다.

2. `selected/portfolio-02-ai-question-flow.png`  
   홈의 AI 질문 카드와 별도 질문 패널 흐름을 보여줍니다. AI agent 관여를 설명할 때 가장 직관적입니다.

3. `selected/portfolio-04-record-ai-structure.png`  
   자연어 기록 입력과 AI 구조화 미리보기 흐름을 보여줍니다. 핵심 기능 설명용으로 좋습니다.

4. `selected/portfolio-05-analysis-report.png`  
   기록 기반 분석 리포트 화면입니다. 저장된 기록을 해석하는 부분을 설명할 때 사용하면 좋습니다.

5. `selected/portfolio-06-shopping-recommendation.png`  
   기록 기반 상품 추천 흐름입니다. 외부 추천/확장 기능 설명에 적합합니다.

6. `selected/portfolio-07-hospital-report.png`  
   병원 상담 준비와 기록 요약 흐름을 보여줍니다. 안전 안내와 병원 연계 포인트를 설명할 때 좋습니다.

7. `selected/portfolio-08-ai-care-panel.png`  
   AI에게 직접 질문하는 패널 화면입니다. 챗봇/케어 질문 기능 강조용입니다.

8. `selected/portfolio-03-home-ai-care.png`  
   홈 화면 단독 컷입니다. 앱 첫인상이나 모바일 UI 소개용으로 사용하면 좋습니다.

## 이번 실행 캡처

루트의 `01-home-ai-care.png`부터 `07-ai-care-panel.png`까지는 현재 로컬 앱을 실행해서 새로 찍은 화면입니다. 다만 현재 실행 세션에서는 프론트가 API 데이터를 요청하지 않아 기록과 알림이 빈 상태로 보였습니다. 그래서 포트폴리오에 바로 넣기 좋은 이미지는 기존 QA 캡처와 구성 이미지를 복사한 `selected` 폴더 기준으로 골랐습니다.
